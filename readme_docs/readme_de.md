"Hyta" ist eine neuartige Python-Bibliothek, die Finanzindikatoren für die technische Analyse bereitstellt, wobei der Schwerpunkt auf inkrementellen Berechnungen liegt. Die inkrementelle Berechnungsfunktion der Bibliothek macht sie besonders geeignet für Echtzeitanwendungen oder jede andere Anwendung, die iterative Eingaben erfordert.

Als relativ neues Projekt ist "hyta" sehr empfänglich für Benutzer-Feedback und Vorschläge zur Verbesserung seiner API, um den besonderen Bedürfnissen der Benutzer gerecht zu werden.

---
