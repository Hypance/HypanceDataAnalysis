"Hyta "是一个新颖的Python库，提供技术分析的金融指标，值得注意的是强调增量计算。该库的增量计算功能使其特别适用于实时应用或任何其他涉及迭代输入的应用。

作为一个相对较新的项目，"hyta "非常乐于接受用户的反馈和建议，以改进其API，满足用户的独特需求。

---
